# Evidence Records

Use this folder to keep audit material for internal validation only.

- `qa-runs/`: smoke and full QA execution records.
- `pilot-runs/`: controlled pilot inputs, outputs, and reviewer feedback.
- `issues/`: observed failures and patch dispositions.
- `release-decisions/`: readiness scorecards and go/no-go records.

Do not mark the agent broadly ready without smoke, QA, pilot, issue-disposition, readiness-scorecard, and go/no-go evidence.
